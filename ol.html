Available options for Service level

RPN More info about RPN Hide

The RPN is an exclusive offer that allows you to use your server's second network interface. This interface is linked to a private network that allows your servers to communicate with each other directly. The RPN also lets you use Online tools like backup via Rsync or NAS/SAN.

Backup (RPN Rsync) More info about Backup (RPN Rsync) Hide

Do backups of your Dedibox servers securely on a synchronised file system, using the protocol Rsync3, an effective protocol and a guarantee for the confidentiality of the transmitted data. Attention: this option does not work without the RPN option.

Backup (FTP) More info about Backup (FTP) Hide

Because you can never be safe from human errors or hardware failure, Online provides a free 100 GB backup space via secure FTP (optionally extensible to 700 GB) to do backups of you critical data for an unlimited period.

Protection against DDoS More info about Protection against DDoS Hide

Attacks by distributed denial of service are now a real threat to your infrastructure's availability. We propose a technical solution, capable of filtering the majority of known DDoS attacks and keep your service available. This option is only available for companies.

Network-attached Storage More info about Network-attached Storage Hide

RPN-SAN is a block storage solution, available over iSCSI via RPN network
